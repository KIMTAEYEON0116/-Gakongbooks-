# 📚 가공독서회 (Gakong Reading Club) 버그 수정 및 개선 이력 보고서 (날짜별 정원)

본 문서는 **가공독서회 (Gakong Reading Club)** 프로젝트의 개발 및 유지보수 과정에서 처리된 모든 디버깅, 버그 수정, UX 개선 내역을 **날짜별(과거 ➔ 최신)**로 체계적으로 정리한 종합 보고서입니다.

## 💡 주요 시스템 건축 및 기술적 성과 요약

> [!NOTE]
> **모듈화 템플릿 구조 (Jinja2 Templates)**
> 메인 뼈대 `index.html`을 중심으로 `components/`(네비게이션, 글로벌 푸터)와 `pages/`(홈, 후보 생성, 도서 상세, 채팅방, 내 서재, 이용 안내 등) 모듈로 체계적으로 분리하여 가독성과 유지보수 편의성을 극대화했습니다.

> [!TIP]
> **온디맨드 REST API 아키텍처**
> 첫 진입 시 무거운 데이터 전체를 로딩하지 않고, 사용자가 클릭하거나 이동하는 시점(온디맨드)에 단일 도서 상세 API(`/api/books/{id}`) 및 아카이브 API를 호출하여 네트워크 트래픽을 아끼고 속도를 극대화했습니다.
